target/socket.h file

Socket targets allow logs to be sent to a Unix domain socket.

Contents

  • Reference
    • Functions

The most common use for socket targets is to send logs to the /dev/log socket or the /var/run/syslog where the syslog daemon is listening.

struct stumpless_target* stumpless_open_socket_target(const char* name, const char* local_socket, int options, int default_facility)

Opens a socket target.

Parameters
name The name of the socket target, and the socket to send logs to.
local_socket The name of the Unix domain socket to connect from. This can be set to NULL if you'd like to use a socket generated by the library.
options The options to use for the logging target. This is a bit-wise or of one or more STUMPLESS_OPTION values.
default_facility The facility code to use for entries that do not have a facility value set on them. This should be a STUMPLESS_FACILITY value.
Returns The opened target if no error is encountered. In the event of an error, NULL is returned and an error code is set appropriately.
